# React TS Note Taking App

This repository contains a clean Note Taking app built using React.js, TypeScript, and Local Storage. It allows users to create, edit, and delete notes, as well as add tags to notes and manage them.

## Features

- Create new notes with a title and content.
- Edit existing notes.
- Delete unwanted notes.
- Add, Delete, Delete tags to notes.
- Save notes and tags locally using the browser's local storage.

## Installation

1. Clone the repository:

```shell
git clone https://github.com/giri-madhan/react-ts-note-taking-app.git
```

2. Navigate to the project directory:

```shell
cd react-ts-note-taking-app
```

3. Install the dependencies:

```shell
npm install
```

4. Start the application:

```shell
npm start
```

5. Open your browser and visit `http://localhost:3000` to see the app running.
